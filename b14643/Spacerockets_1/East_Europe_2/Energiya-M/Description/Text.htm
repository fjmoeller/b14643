Energiya-M
 
 

RKK Energiya
 

For some time now, RKK Energiya has been working on reviving the Energiya launcher. The aim is to adapt the design to payloads of up to 40 t for a LEO or 6.0 t for a GEO. The core is to be radically changed under the name Energiya-M. Only two boosters will be added on the sides. The fuel mass of the core is to be reduced to 234 t. Only a single RD-0120 engine will be installed for combustion. A cryogenic upper stage with a 100 kN engine and 70 t LH2/LOX as fuel is to be re-ignitable. So far, only an engineering model has been built.
